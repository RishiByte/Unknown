# 🍽️ L'ATELIER – Smart Recipe & Kitchen Assistant

L'ATELIER is an AI-powered kitchen assistant designed to help users discover recipes, manage ingredients, and optimize their cooking experience. Built with a modern UI and powered by TheMealDB API, it delivers personalized recipe recommendations based on user preferences and available inventory.

---

## 🚀 Features

- 🧠 **Smart Recipe Recommendations**
  - Suggests dishes based on available ingredients and user ratings

- 📦 **Inventory Management**
  - Track ingredients you currently have
  - Helps reduce food waste

- 📖 **Recipe Explorer**
  - Fetches recipes dynamically using TheMealDB API
  - View ingredients, instructions, and categories

- ⭐ **Rating System**
  - Rate recipes to improve future recommendations

- 🔥 **Trending Recipes**
  - Discover globally popular dishes

- 💡 **Pro Tips**
  - Smart cooking tips to enhance your meals

- 📊 **Dashboard Analytics**
  - View saved recipes, ratings, and inventory stats
